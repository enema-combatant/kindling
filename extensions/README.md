# Extensions

After completing the core missions, these extensions take you deeper into specific areas.

| Extension | What You'll Learn | Prerequisites |
|-----------|------------------|---------------|
| [Fine-Tuning](fine-tuning/) | Train a model on your own data | Mission 04 + GPU or cloud GPU |
| [Security Hardening](security-hardening/) | TLS, authentication, container isolation | Missions 00-02 |
| [Monitoring](monitoring/) | Prometheus metrics, health checks, Grafana | Missions 00-02 |
| [Multi-Node](multi-node/) | Distribute inference across machines | Missions 00-02 + extra hardware |
| [Field Deployment](field-deployment/) | Offline operation on laptop/phone | Missions 00-05 |
| [Mesh Sync](mesh-sync/) | Offline synchronization via LoRa/mesh | Field Deployment + radio hardware |

Extensions are currently stubs with roadmaps. They'll be fleshed out based on community interest.
